MindFlow Windows 便携版

启动方法
1. 先完整解压下载的 ZIP 文件。
2. 打开解压后的 MindFlow 文件夹。
3. 双击根目录里的 MindFlow.exe。

文件说明
- MindFlow.exe：软件启动入口。
- app 文件夹：软件运行组件，请勿删除、改名或单独移动。

本版本不需要安装，也不需要绑定账号。
笔记、任务、画图、工作流和思维导图数据均保存在本机。
笔记和任务提醒会在 MindFlow 运行时发送 Windows 本地通知。

桌面任务挂件
- 在“设置 > 任务挂件”中点击“显示任务挂件”。
- 可开启“开机启动任务挂件”；登录 Windows 后只显示挂件，不打开主窗口。
- 挂件关闭按钮会隐藏窗口，仍可从系统托盘重新打开或彻底退出。
- 开机启动依赖当前 MindFlow 文件位置，请勿在开启后移动或删除整个软件文件夹。
